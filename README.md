# bird-flocking-simulation-unity
This project is a real-time flocking simulation built in Unity using C#.
It demonstrates basic autonomous agent behavior inspired by the Boids model, where multiple agents (birds) move collectively based on simple rules.

The system currently implements:
Seek behavior (toward mouse position)

Separation behavior (collision avoidance)
Features:
-Multi-agent simulation (configurable number of birds)

-Real-time movement using physics-inspired logic

-Mouse interaction (birds follow cursor)

-Separation system to prevent overlap

-Smooth motion using velocity and acceleration
